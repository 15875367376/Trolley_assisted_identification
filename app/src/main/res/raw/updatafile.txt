更新时间：2023.12.28
应用版本：安卓辅助识别6.0（当前已安装）
更新内容：
    1. 更新为支持Android Studio Dolphin | 2022.1.1 Patch 2 版本开发工具的项目工程，当前应用的项目工程可使用新版本IDE进行编辑修改，旧版本的Android Studio（下称AS）因为种种原因，现已不适合继续使用，请大家尽快更新AS继续学习和开发。
    2. 更新AS后，若旧版本工程若没有重新配置过gradle，请跟着下面步骤进行：
    安装完成新版本AS → 打开项目工程 → 点击左上角File → 点击Project Structure → 点击左侧Project选项 → 将右侧内容更改为如下版本：
    Android Gradle Plugin Version ↓↓
    4.2.2
    Gradle Version ↓↓
    6.7.1
    3.有TFT车牌识别功能模块，使用开源的HyperLPR3识别库进行识别。        这是车牌识别1.0版本
    4.有红绿灯识别模块，是用yolov5框架进行训练的模型并移植到安卓系统中
    5.有二维码识别模块，OpenCv + 二维码识别处理库做的二维码识别
    6.有TFT交通标志物识别模块，是用yolov5框架进行训练的模型并移植到安卓系统中
    7.有了TFT车型识别模块，是用yolov5框架进行训练的模型并移植到安卓系统中
    8.有了TFT口罩行人识别模块，是用yolov5框架进行训练的模型并移植到安卓系统中
    9.有TFT车牌识别功能模块，使用是用yolov5框架进行训练的模型并移植到安卓系统中 + 谷歌的OCR英文包做车牌识别。     这是车牌识别2.0版本
    10.有TFT车牌识别功能模块，使用谷歌的OCR中文包 + OpenCv做车牌识别。           这是车牌识别3.0版本
    11.有TFT车牌识别功能模块，使用百度飞桨OCR + OpenCv做车牌识别。             这是车牌识别4.0版本
    12.有TFT文字识别功能模块，使用谷歌的OCR中文包 + OpenCV做文字识别           这是文字识别1.0版本
    13.有TFT文字识别功能模块，使用百度飞桨OCR + OpenCV做文字识别              这是文字识别2.0版本
    14.后续的TFT的各种识别后续更新
    15.有什么问题请联系开发者
    修改完成后保存回到代码编辑页面进行同步/编译即可
